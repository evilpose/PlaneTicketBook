# planeticketbook

## 项目安装准备
```
npm install
```

### 编译和热重新加载以进行开发
```
npm run serve
```

### 编译和缩小以供生产上线
```
npm run build
```

### 运行单元测试
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
